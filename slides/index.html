¿Qué es?

DataTables es un plug-in para la librería jQuery de Javascript. Es probablemente la herramienta más flexible para cargar tablas de datos en una página web y permite al usuario un alto nivel de interacción.

¿Cómo usarlo?

  1. instalar git
  2. git pull del repositorio: 
    > git clone git@github.com:miguelfg/jpd16-workshop-datatables.git
  3. instalar servidor local
    • python 2.X: 
      > python -m SimpleHTTPServer 8000
    • python 3.X: 
      > python -m http.server 8000
    • npm
  4. ejecutar servidor local desde la raíz del repositorio
  5. visitar url: http://localhost:xxx/src/example-1/
  6. [OPT] instalar csvkit
